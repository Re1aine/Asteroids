# Asteroids
A classic arcade-style Asteroids game built in Unity.

## 🕹️ Controls
| Action | Key |
|--------|-----|
| **Shoot Bullet** | `SPACE` (Press) |
| **Shoot Laser** | `SHIFT` (Hold) |

## 📥 Setup
1. Install `Unity 6000.0.34f1`  
2. Open `Gameplay` scene
3. Play
